# ColdChain-QA 前端

Vue3 + Vite 冷链物流智能问答界面，支持 SSE 流式对话、知识库管理、数字人助手。

## 快速启动

```bash
# 确保后端已运行在 localhost:8000
cd frontend
npm install
npm run dev
```

访问 http://localhost:5173

## 功能

- SSE 流式问答（打字机效果）
- Base / LoRA 模型切换
- 知识库文档上传 / 删除
- 引用来源实时展示
- 回答反馈（👍/👎）
- 数字人助手动画
